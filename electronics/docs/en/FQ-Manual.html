Creating a new circuit

After clicking on the button or menu item, the diagram and the associated data in the right-hand window area are cleared. First, enter a name for the new circuit to be calculated. Below this, enter the list of components and their position in the form of numbered nodes (click on the New or Edit buttons on the right-hand side).
Further details on circuit input and range settings can be found below.

Load a saved circuit description

A previously saved circuit (NWD file) is loaded, the diagram is calculated and displayed. To change component properties, first select the desired component and then click on the Edit button. In the area at the bottom right, you can set the scaling for the Bode plot and the Nyquist plot, as well as the line properties (color, style, etc.).

Save circuit

The circuit data and scaling are stored in a file and can be reloaded and displayed at any time again.

Save the circuit as new file

This saves an existing and possibly modified circuit under a different name in a file.

List of recently loaded circuits

The circuit menu always contains a list of the most recently loaded circuits. Clicking on a file name loads and displays the corresponding circuit.

Quit program

The program is terminated. If changes have been made to the circuit beforehand, you will be asked if you want to save them.

Recalculate diagram

The circuit is recalculated and the result is displayed.

Copy diagram to clipboard

The current diagram (Bode or Nyquist plot) is copied to the clipboard and can be pasted as an image in other applications.

Export diagram to a Windows meta file

The diagram is saved as a meta file. This format can be read by many programs in order to insert the image there.

Print diagram

The current diagram (Bode or Nyquist plot) is printed on the selected printer.

Directory with circuit files

Here you can select the directory in which the circuits are to be saved.

Select printer

Here you can select the printer on which the diagrams are to be printed.

Enter circuit description

Before entering a circuit, you should first draw it on paper and number the components and nodes (see also the examples below). Up to 50 components and 25 nodes are possible. The reference point (ground) always receives node number 0, and the input preferably receives node number 1. All other nodes can be labeled as desired.

Components:

Then use the New button to enter the properties of the components one after the other. For resistors, capacitors, and inductors, the following information must be provided:

  • Number of component (e.g. R1, C2 oder L3)
  • Value of component (e.g. 10k)
  • Connections (Numbers of the two connection nodes)

An additional component description can be entered in the Note field. When entering values, the units are predetermined by the type of component (Ohm for resistors, Farad for capacitors, and Henry for inductor). The usual prefixes (µ, m, k, M, etc.) can be used for the numerical values. By clicking on the button on the far right, a value can be selected from a standard series (e.g. E12 or E24).

In addition, additional values (e.g., different resistances) can be entered for any selected passive component (Variable button). This allows you to check the effect of changes to this component on the frequency response. The curves of the additional values are shown as dashed lines in the diagram..

Slightly different specifications are required for operational amplifiers:

  • Number of component (e.g.. OP1)
  • Gain of the OP (default = 106)
  • Cutoff frequency of the OP (default = 20 Hz)
  • Connections (Numbers of the two inputs +/- and the output)

Circuit:

Next, select the input node (usually No. 1) and one or more output nodes (max. 4) for which the frequency responses are to be calculated. Finally, click on the Show button. The frequency responses of amplitude and phase are calculated for the preset frequency range of 10 Hz to 100 kHz. The scaling for amplitude and phase is done automatically. Frequency and amplitude are displayed on a logarithmic scale.

Automatic scaling can be disabled for each curve to allow for individual settings. The color and style of the displayed curves can also be customized. For each output node, there is a curve (solid line) for the main value and, if selected, additional dashed curves for the variable values.


Selection of standard values

Commercially available components are offered in values that correspond to the internationally standardized values of the E series. To facilitate the entry of such a value, the dialog box shown on the right can be called up in the program. The E6, E12, E24, and E24 series with values from E48 are supported.

The desired value is easily compiled by clicking on the self-explanatory selection options.


Variable values for a selected component

Additional values (e.g. different resistances) can be entered for any selected passive component. This allows you to check the effect of changes to this component on the frequency response. The curves for the additional values are shown as dashed lines in the diagram.

The buttons can be used to add individual values or values from a standard series or to change selected values. Values are automatically sorted by size when the dialog box is closed.

Set ranges and scaling

First, set the desired frequency range for the Bode plot and the scaling of the frequency axis. This setting also applies to the display of the Nyquist plot.

The ranges for amplitude and phase are normally adjusted by the program to achieve optimal display. This automatic function can be switched off if other settings are desired. For amplitude, the scaling can be linear, logarithmic, or in dB. The range settings must be adjusted accordingly. The phase is always displayed on a linear scale.

Automatic scaling can also be disabled for the Nyquist plot. If the curve is displayed as angular, this is because the frequency intervals were too large during the calculation. Unfortunately, this cannot yet be adjusted in the current version.


Settings for the lines

The color, width, and style of the lines can be customized. The settings for the variable values refer to the option described above for specifying several additional values for a component.
