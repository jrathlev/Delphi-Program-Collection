## Delphi translations

This repository contains *po* and *mo* files for the translations processed by [GnuGetText for Delphi](https://github.com/jrathlev/GnuGetText-for-Delphi). 

### Supported libraries:
- Delphi 10 Seattle
- Indy 10

### Supported languages:
- German (complete)
- Turkish (complete)
- Ukrainian (complete)
- Chinese (complete)
- French (71 %)
- Dutch (67%)
- Italian (48%)-
- Russian (44%) 
- Hungarian (34%)
- Portuguese (31%)
- Spanish (22%)
- Polish (0%)
