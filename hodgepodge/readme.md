## Some helpful utility programs

### VCL programs
- **MonCheck:** Show properties of all connected monitors 

### Console programs
- **ChangeModHint:** Change "last modified" hint in source files
- **KnownFolders:** Show a list of all known folders
- **PowerCaps:** Show the power capabilities of your computer
- **ShowReparse:** Show a list of all reparse points in a directory
- **ShowSession:** Show session data for the logged-in user
- **ShowLangSettings:** Show the settings for region and language
- **ShowVol:** Show all available volumes and their properties
- **WinKey:** Show the product key of the installed Windows system
