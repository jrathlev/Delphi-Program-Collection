## Delphi program collection

This repository contains a collection of several **programs**, **components**, **dialogs** and **units**, all written in [Delphi](https://en.wikipedia.org/wiki/Delphi_(software)). Most of them support multiple languages using the [GnuGettext for Delphi package](https://github.com/jrathlev/GnuGetText-for-Delphi). The translations for several languages are included.

**Acknowledgements:**  Some programs are using routines from the following sources
- **SVGIconImageList** package by [Ethea](https://ethea.it/docs/svgiconimagelist)
- **Image32** library by [Angus Johnson](https://angusj.com/image32/Docs/_Body.htm)
